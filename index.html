Last Updated：March 26, 2017

DoLauncher ("we," "us," or "our") collects information from you in order to provide better service and a better user experience. Your privacy is a top priority to us, and this Privacy Policy explains how we collect, use, and protect your information.

What information we use and how we use it:

We may collect and use non-personal information in the following circumstances. To have a better understanding in user’s behavior, solve problems in products and services, improve our products, services and advertising, we may collect non-personal information such as the data of install, click of app, installed app list, frequency of use, country, equipment and channel, etc.

"Non-personal information" is data in a form that does not permit direct association with any specific individual, such as your Android ID, CPN model, memory size, your phone IMEI number, phone model, rom, phone operator, location, install, uninstall, frequency of use, etc.

Following are the explaination for the used permissions:

1. read phone status and identity (android.permission.READ_PHONE_STATE)

This permission allows DoLauncher to recognize an incoming call and to switch between the launcher interface and call interface. and it allow DoLauncher to read aid for ad

2. access fine location

This permission is used to display weather info in your location, NO location info are tracked and stored in remote server.

3. modify or delete the contents of your USB storage

This permission allows users to delete DoLauncher wallpaper stored on their SD card.

4. read the contents of your USB storage

This permission allows users to view DoLauncher wallpaper stored on their SD card.

5. read Google service configuration

This information is used to acquire advertising ID. DoLauncher provides users with better advertising service by using such anonymous ID.

6. change network connectivity

This permission is used in DoLauncher settings and notification toolbar, in order to change network connectivity.

7. connect and disconnect from Wi-Fi

This permission is used in DoLauncher settings and notification toolbar in order to connect and disconnect from Wi-Fi.

8. full network access

This permission is used to access the device’s network for certain functions including receiving launcher update notifications or accessing app classification labels.

9. view network connections

This permission is used in DoLauncher settings and notification toolbar, in order to view network connections.

10. view Wi-Fi connections

This permission is used in DoLauncher settings and notification toolbar, in order to view Wi-Fi connections.

11. access Bluetooth settings

This permission is used in DoLauncher settings, in order to turn on and off the Bluetooth.

12. pair with Bluetooth devices

This permission is used in DoLauncher settings, in order to pair with Bluetooth devices.

13. close other apps

This permission is used in DoLauncher Boost in order to turn off the back-end apps and make the phone run faster.

14. retrieve running apps

This permission is used in DoLauncher Boost in order to view the running apps.

15. run as startup

DoLauncher will run as startup to provide users with a launcher service.

16. control vibration

This permission allows DoLauncher to make the phone vibrate once after users set the device on vibrate, confirming the setting has been turned on.

17. adjust your wallpaper size

This permission allows users to crop DoLauncher wallpaper to the desired size.

18. set wallpaper

This permission allows users to set a selected DoLauncher image as their wallpaper.

19. read sync settings

This permission is used in DoLauncher power’s save mode, and allows DoLauncher to recognize whether sync settings are turned on or off.

20. toggle sync on and off

This permission is used in DoLauncher power’s save mode, and allows DoLauncher to synchronize with the user's sync settings.

21. expand/collapse status bar

This permission is used for the gesture feature of DoLauncher to expand and collapse the status bar.

22. install shortcuts

This permission is used to install shortcuts on other launchers, our users can continue to use DoLauncher related functions.

23 measure app storage space

This permission is used to acquire the amount of storage space used by an application.

24. modify system settings

This permission is used in DoLauncher settings, in order to switch or adjust ringtone, vibration and brightness level of the screen.

25. read Home settings and shortcuts

This permission is used to acquire the user’s Home settings and shortcuts, in order to maintain user preferences.

26. uninstall shortcuts

This permission allows DoLauncher to delete the shortcuts created by DoLauncher in other Android launchers.

27. draw over other apps

This permission allows the DoLauncher sidebar to be displayed as a floating window on other apps.
